# تحذير Source Map (يمكن تجاهله)

التحذير التالي قد يظهر عند التشغيل أو البناء:

```
Failed to parse source map from '...node_modules/stylis-plugin-rtl/src/stylis-rtl.ts'
```

**السبب:** حزمة `stylis-plugin-rtl` تشير إلى ملف مصدر غير موجود في النسخة المنشورة.

**التأثير:** لا يوجد. البناء يكتمل بنجاح والتطبيق يعمل بشكل طبيعي. التحذير لا يؤثر على الإنتاج.

**إخفاء التحذير (اختياري):** إذا أردت إخفاءه في بناء الإنتاج فقط، يمكنك إضافة في ملف `.env` أو سكربت البناء:
```
GENERATE_SOURCEMAP=false
```
(هذا يعطّل كل خرائط المصدر في الإنتاج، وليس تحذيراً فقط.)
